pre-reqs for challenge-1
========================
Create a Security group with below inbound rules for jenkins instance in the default vpc
	22 from your IP
	80 from anywhere
	8080 from anywhere

IAM role for Jenkins instance with right persmissions

Run the below command from the cloudshell

aws ec2 run-instances \
> --image-id ami-089ffe32605122764 \
> --count 1 \
> --instance-type t2.micro \
> --security-group-ids sg-0aca51b53876e08a5 \
> --iam-instance-profile Name=EC2JenkinsAdminRole \
> --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=jenkins-from-cli}]' \
> --user-data file://script.sh \
> --region us-west-2

aws ec2 run-instances --image-id ami-089ffe32605122764 --count 1 --instance-type t2.large --security-group-ids sg-0aca51b53876e08a5 --iam-instance-profile Name=EC2JenkinsAdminRole --tag-specifications 'ResourceType=instance,Tags=[{Key=Name,Value=jenkins-from-cli}]' --user-data file://script.sh --region us-west-2

aws s3api create-bucket --bucket terraform-remote-state-12354 --create-bucket-configuration LocationConstraint=us-west-2

Wait for the EC2 to get 2/2 status check and then hit the below endpoint
public-ip:8080

Configure the Jenkins with default values
plugins to be installed "Cloudbees AWS Credentials" & "Terraform"
add terraform tool in global tool configuration
